# Кастомный префикс на стилях

Чтобы сделать версию с кастомным префиксом (не `cdek`) необходимо запускать джобу `publish-with-prefix` вместо `publish`.
Эта джоба требует переменную `STYLE_PREFIX`. Задавайте ее нажимая на название джобы вместо кнопки `Run`.

В `STYLE_PREFIX` необходимо написать желаемый префикс. Джоба изменит название версии из `package.json` и добавит в конец `-{ваш префикс}`, так что
вам не нужно как-то помечать версию вручную.
