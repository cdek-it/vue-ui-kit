# Стилизация

### Переменные и миксины

Библиотека предоставляет для импорта готовые переменые SCSS c цветами и миксины для шрифтов и медиа запросов.

Для подключения добавьте себе в какой-нибудь scss файл:

```css dark
@import '@cdek-ui-kit/vue/scss';
```

Эта команда автоматически подтянет вам все переменные в этот файл.

#### Colors

Цвета взяты с [фигмы](https://www.figma.com/file/ZIhkqRfKAFAf3w06aqfWzz/CDEK-Web-Library?node-id=2270%3A7432&mode=dev), названы ровно также как там.
Проще всего смотреть на макете название цвета и брать последнее слово после /.
Каждый цвет привязан с соответствующей CSS переменной с тем же названием, для переопределения просто добавьте эти переменные в `:root`, тогда вы
сможете сделать другую цветовую палитру.

#### Медиа

Медиа представлены как для desktop-first, так и для mobile-first подхода. Также есть дополнительное медиа правило для hover эффектов, настоятельно
рекомендую его использовать, чтобы у вас не возникали странные эффекты на тач устройствах.

<details>
  <summary>media.scss</summary>

```css dark
// Desktop-first

@mixin media-xs {
  @media (max-width: 576px) {
    @content;
  }
}

@mixin media-sm {
  @media (max-width: 768px) {
    @content;
  }
}

@mixin media-md {
  @media (max-width: 992px) {
    @content;
  }
}

@mixin media-lg {
  @media (max-width: 1200px) {
    @content;
  }
}

@mixin media-xl {
  @media (max-width: 1400px) {
    @content;
  }
}

// Mobile first

@mixin media-min-sm {
  @media (min-width: 576px) {
    @content;
  }
}

@mixin media-min-md {
  @media (min-width: 768px) {
    @content;
  }
}

@mixin media-min-lg {
  @media (min-width: 992px) {
    @content;
  }
}

@mixin media-min-xl {
  @media (min-width: 1200px) {
    @content;
  }
}

@mixin media-min-xxl {
  @media (min-width: 1400px) {
    @content;
  }
}

// Hover

@mixin media-hover {
  @media (hover: hover) {
    &:hover {
      @content;
    }
  }
}
```
