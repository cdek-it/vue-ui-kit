<Controls of={ToastStories.WithContent} />

### Width
Ширина тоста задаётся через CSS-класс на компоненте. Доступные значения:
<details>
  <summary>Примеры кода</summary>

- `sm` (20rem)
```html
<Toast class="p-toast-sm" />
```
- `base` (25rem, по умолчанию)
```html
<Toast />
```
- `lg` (30rem)
```html
<Toast class="p-toast-lg" />
```
- `xlg` (45rem)
```html
<Toast class="p-toast-xlg" />
```

</details>
<Canvas of={ToastStories.Width} />

#### Аргументы и события
<Controls of={ToastStories.Width} />
