# Toast

[PrimeVue Toast](https://primevue.org/toast), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=15953-4478)

`Toast` — всплывающее окно, которое используется для отображения сообщений. Подобно уведомлениям в iOS и Android.

<br />
<hr />
<br />

### Basic

Подключение `Toast` осуществляется через хук `usePBlockToast` и вызов метода `.add()` с конфигом.

```html dark
<!-- Ваш компонент -->
<script setup lang="ts">
  import PBlockToast from '@/primeBlocks/PBlockToast/PBlockToast.vue'
  import { usePBlockToast } from '@/primeBlocks/PBlockToast/usePBlockToast';

  const toast = usePBlockToast();

  const onClickToastShowBtn = () => {
    toast.add({
      severity: 'success', // Тип Toast
      summary: 'Заголовок сообщения',
      detail: 'Дополнительная информация',
      life: 3_000, // Время, в течении которого будет показан Toast. В миллисекундах
      closable: true, // Закрываемый или нет. Значение `true` по умолчанию.
    });
  };
</script>

<template>
  <div>
    <PBlockToast />
    <Button label="Показать Toast" @click="onClickToastShowBtn" />
  </div>
</template>
```

### Toast's Types

`Toast` имеет 4 типа:
* `success` — успех;
* `info` — информация;
* `warn` — предупреждение;
* `error` — ошибка.

Тип указывается в поле `severity`.

### Custom Container

`Toast` с пользовательским шаблоном контейнера и стилизованной кнопкой закрытия.

```html dark
<!-- Ваш компонент -->
<script setup lang="ts">
  import {
    usePBlockToast,
    PBlockToastMessageIcon,
  } from '@/primeBlocks/PBlockToast/usePBlockToast';
  import PBlockToast from '@/primeBlocks/PBlockToast/PBlockToast.vue';

  const toast = usePBlockToast();

  const onClickToastShowBtn = () => {
    toast.add({
      severity: 'success',
      summary: 'Заголовок сообщения',
      detail: 'Дополнительная информация',
      life: 3_000,
      icon: PBlockToastMessageIcon.success, // ti-circle-check
    });
  };
</script>

<template>
  <div>
    <PBlockToast>
      <template #container="{ message, closeCallback }">
        <div class="p-toast-message-content">
          <div class="p-toast-accent-line"></div>
          <i :class="'p-toast-message-icon ti ' + message.icon"></i>
          <div class="p-toast-message-text">
            <span class="p-toast-summary">{{ message.summary }}</span>
            <div class="p-toast-message-caption">{{ message.detail }}</div>
          </div>
          <Button
            class="p-toast-close-button"
            icon="ti ti-x"
            variant="text"
            rounded
            @click="closeCallback"
          />
        </div>
      </template>
    </PBlockToast>
    <Button label="Показать Toast" @click="onClickToastShowBtn" />
  </div>
</template>
```

### Custom Container & Content

`Toast` с:
* пользовательским шаблоном контейнера;
* с дополнительным содержимым ячеек под подписью;
* стилизованной кнопкой закрытия.

```html dark
<!-- Ваш компонент -->
<script setup lang="ts">
  import {
    usePBlockToast,
    PBlockToastMessageIcon,
  } from '@/primeBlocks/PBlockToast/usePBlockToast';
  import PBlockToast from '@/primeBlocks/PBlockToast/PBlockToast.vue';

  const toast = usePBlockToast();

  const onClickToastShowBtn = () => {
    toast.add({
      severity: 'success',
      summary: 'Заголовок сообщения',
      detail: 'Дополнительная информация',
      life: 3_000,
      icon: PBlockToastMessageIcon.success,
      content: 'Контент',
      cells: ['Ячейка 1', 'Ячейка 2'],
    });
  };
</script>

<template>
  <div>
    <PBlockToast>
      <template #container="{ message, closeCallback }">
        <div class="p-toast-message-content">
          <div class="p-toast-accent-line"></div>
          <i :class="'p-toast-message-icon ti ' + message.icon"></i>
          <div class="p-toast-message-text">
            <span class="p-toast-summary">{{ message.summary }}</span>
            <div class="p-toast-message-caption">{{ message.detail }}</div>
            <div class="mt-4">
              <div class="text-sm">{{ message.content }}</div>
            </div>
            <div class="flex gap-2 mt-2">
              <div
                v-for="(cell, idx) in message.cells"
                :key="idx"
                class="text-sm"
              >{{ cell }}</div>
            </div>
          </div>
          <Button
            class="p-toast-close-button"
            icon="ti ti-x"
            variant="text"
            rounded
            @click="closeCallback"
          />
        </div>
      </template>
    </PBlockToast>
    <Button label="Показать Toast" @click="onClickToastShowBtn" />
  </div>
</template>
```
