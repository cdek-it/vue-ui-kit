# Carousel

[PrimeVue Carousel](https://primevue.org/carousel), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=1187-1633&t=j8q05RFyXpKxntpS-11)

### Basic

Базовый пример Carousel с горизонтальной прокруткой.

```html dark
<Carousel :value="products" :numVisible="3" :numScroll="1">
  <template #item="slotProps">
    <div style="border: 1px dashed #e2e2e4; border-radius: 10.5px; padding: 20px; text-align: center;">
      <div style="font-size: 14px; color: #1dc831; font-weight: 600;">{{ slotProps.data.name }}</div>
    </div>
  </template>
</Carousel>
```
