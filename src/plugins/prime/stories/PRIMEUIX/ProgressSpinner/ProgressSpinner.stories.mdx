# ProgressSpinner

ProgressSpinner - это компонент индикатора загрузки процесса.

[Документация PrimeVue](https://primevue.org/progressspinner/)

---

## Базовый

ProgressSpinner используется с базовыми настройками по умолчанию.

<Story name="Базовый">
  {BasicTemplate.bind({})}
</Story>

```html
<ProgressSpinner />
```

---

## Пользовательский

ProgressSpinner можно настроить с помощью различных стилей и свойств.

<Story name="Пользовательский">
  {CustomTemplate.bind({})}
</Story>

```html
<ProgressSpinner 
  style="width: 50px; height: 50px" 
  strokeWidth="8" 
  fill="var(--p-surface-ground)" 
  animationDuration=".5s" 
/>
```
