# Password

Password - это поле ввода для паролей с индикатором надежности и переключением видимости.

[Документация PrimeVue](https://primevue.org/password/)

---

## Базовый

Password используется с двусторонней привязкой через v-model.

<Story name="Базовый">
  {BasicTemplate.bind({})}
</Story>

```html
<Password v-model="value" placeholder="Пароль" />
```

---

## Индикатор надежности

Свойство `toggleMask` добавляет кнопку для переключения видимости пароля.

<Story name="Индикатор надежности">
  {MeterTemplate.bind({})}
</Story>

```html
<Password v-model="value" placeholder="Пароль" toggleMask />
```

---

## Без переключения видимости

Свойство `feedback` отключает индикатор надежности пароля.

<Story name="Без переключения видимости">
  {WithoutToggleTemplate.bind({})}
</Story>

```html
<Password v-model="value" :feedback="false" placeholder="Пароль" />
```

---

## Шаблон

Слоты `header` и `footer` позволяют добавить пользовательский контент.

<Story name="Шаблон">
  {TemplateCustomTemplate.bind({})}
</Story>

```html
<Password v-model="value" placeholder="Пароль" toggleMask>
  <template #header>
    <div class="font-semibold">Требования к паролю</div>
  </template>
  <template #footer>
    <ul>
      <li>Минимум 8 символов</li>
      <li>Одна заглавная буква</li>
    </ul>
  </template>
</Password>
```

---

## Заполненный

Вариант `filled` применяет альтернативный стиль оформления.

<Story name="Заполненный">
  {FilledTemplate.bind({})}
</Story>

```html
<Password v-model="value" variant="filled" placeholder="Пароль" />
```

---

## Отключенный

Отключенное состояние предотвращает взаимодействие с полем.

<Story name="Отключенный">
  {DisabledTemplate.bind({})}
</Story>

```html
<Password v-model="value" disabled />
```

---

## Невалидный

Невалидное состояние используется для индикации ошибок валидации.

<Story name="Невалидный">
  {InvalidTemplate.bind({})}
</Story>

```html
<Password v-model="value" invalid placeholder="Пароль" />
<small class="text-red-600">Пароль обязателен</small>
```
