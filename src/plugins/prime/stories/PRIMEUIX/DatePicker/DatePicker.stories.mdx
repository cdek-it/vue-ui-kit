# DatePicker

DatePicker - это компонент для выбора даты и времени.

[Документация PrimeVue](https://primevue.org/datepicker/)

---

## Базовый

DatePicker используется с двусторонней привязкой через v-model.

<Story name="Базовый">
  {BasicTemplate.bind({})}
</Story>

```html
<DatePicker v-model="date" placeholder="Выберите дату" />
```

---

## Формат

Свойство `dateFormat` определяет формат отображения даты.

<Story name="Формат">
  {FormatTemplate.bind({})}
</Story>

```html
<DatePicker v-model="date" dateFormat="dd.mm.yy" placeholder="ДД.MM.ГГ" />
```

---

## Иконка

Свойство `showIcon` добавляет иконку календаря.

<Story name="Иконка">
  {IconTemplate.bind({})}
</Story>

```html
<DatePicker v-model="date" showIcon iconDisplay="input" placeholder="Выберите дату" />
```

---

## Диапазон

Режим `range` позволяет выбрать диапазон дат.

<Story name="Диапазон">
  {RangeTemplate.bind({})}
</Story>

```html
<DatePicker v-model="dates" selectionMode="range" placeholder="Выберите диапазон" />
```

---

## Множественный выбор

Режим `multiple` позволяет выбрать несколько дат.

<Story name="Множественный выбор">
  {MultipleTemplate.bind({})}
</Story>

```html
<DatePicker v-model="dates" selectionMode="multiple" placeholder="Выберите даты" />
```

---

## Время

Свойство `showTime` добавляет выбор времени.

<Story name="Время">
  {TimeTemplate.bind({})}
</Story>

```html
<DatePicker v-model="datetime" showTime hourFormat="24" placeholder="Дата и время" />
```

---

## Только время

Свойство `timeOnly` отображает только выбор времени.

<Story name="Только время">
  {TimeOnlyTemplate.bind({})}
</Story>

```html
<DatePicker v-model="time" timeOnly placeholder="Выберите время" />
```

---

## Месяц

Режим просмотра `month` позволяет выбрать месяц.

<Story name="Месяц">
  {MonthTemplate.bind({})}
</Story>

```html
<DatePicker v-model="date" view="month" dateFormat="mm/yy" placeholder="Выберите месяц" />
```

---

## Год

Режим просмотра `year` позволяет выбрать год.

<Story name="Год">
  {YearTemplate.bind({})}
</Story>

```html
<DatePicker v-model="date" view="year" dateFormat="yy" placeholder="Выберите год" />
```

---

## Отключенный

Отключенное состояние предотвращает взаимодействие с компонентом.

<Story name="Отключенный">
  {DisabledTemplate.bind({})}
</Story>

```html
<DatePicker v-model="date" disabled />
```

---

## Невалидный

Невалидное состояние используется для индикации ошибок валидации.

<Story name="Невалидный">
  {InvalidTemplate.bind({})}
</Story>

```html
<DatePicker v-model="date" invalid placeholder="Выберите дату" />
<small class="text-red-600">Дата обязательна</small>
```
