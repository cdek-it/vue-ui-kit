# Card

[PrimeVue Card](https://primevue.org/card), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=1213-4134)

Card компонент используется для группировки контента с заголовком, подзаголовком и действиями.

## Полная карточка

Карточка со всеми элементами: header, title, subtitle, content и footer.

<Story name="Default">{Template.bind({})}</Story>

```html dark
<PrimeCard>
  <template #header>
    <div class="bg-gray-100 h-32 flex items-center justify-center text-gray-400">
      <i class="ti ti-photo text-3xl"></i>
    </div>
  </template>
  
  <template #title>
    Title
  </template>
  
  <template #subtitle>
    Caption
  </template>
  
  <template #content>
    <div class="text-sm">
       Card content goes here. This is a flexible area for any content.
    </div>
  </template>
  
  <template #footer>
    <Button label="Action" />
  </template>
</PrimeCard>
```

## Overlay варианты

Карточка может отображаться с тенью (overlay: true) или без тени (overlay: false).

<Story name="Overlay">{OverlayTemplate.bind({})}</Story>

```html dark
<!-- С тенью -->
<PrimeCard :pt="{ root: { class: 'shadow-md' } }">
  <!-- content -->
</PrimeCard>

<!-- Без тени -->
<PrimeCard :pt="{ root: { class: 'shadow-none' } }">
  <!-- content -->
</PrimeCard>
```
