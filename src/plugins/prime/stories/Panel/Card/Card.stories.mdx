# Card

[PrimeVue Card](https://primevue.org/card), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/%F0%9F%A7%A9-UI-kit-PrimeFace--1.0.0-beta-?node-id=1213-4134)

Card компонент используется для группировки контента. Поддерживает header, title, subtitle, content и footer слоты.

## Варианты

### Default (Overlay)

Карточка с тенью (overlay: true) - используется по умолчанию.

```html dark
<Card>
  <template #title>Title</template>
  <template #subtitle>Caption</template>
  <template #content>
    Content
  </template>
</Card>
```

<Story name="Default">{Template.bind({})}</Story>

### Inline (без тени)

Карточка без тени (overlay: false) - для использования внутри других контейнеров.

```html dark
<Card :pt="{ root: { class: 'p-card-inline' } }">
  <template #title>Title</template>
  <template #subtitle>Caption</template>
  <template #content>
    Content
  </template>
</Card>
```

<Story name="Inline" args={{inline: true}}>{Template.bind({})}</Story>
