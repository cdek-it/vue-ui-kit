# Accordion
[PrimeVue Accordion](https://primevue.org/accordion/), [Макет](<https://www.figma.com/design/4TYeki0MDLhfPGJstbIicf/UI-kit-PrimeFace--DS-?node-id=1207-3420&m=dev>)

## Общая информация

Для настройки работающей группы панелей, необходимо подключение:
```ts dark
import Accordion from 'primevue/accordion';
import AccordionPanel from 'primevue/accordionpanel';
import AccordionHeader from 'primevue/accordionheader';
import AccordionContent from 'primevue/accordioncontent';
```

`Accordion`**\*** - Общий контейнер.  В нём располагаются от 1-й до N панелей, который раскрываются/сворачиваются.

`AccordionPanel` - Раскрывающаяся/сворачивающаяся панель. Атрибут `value`**\*\*** - идентификатор панели.

`AccordionHeader` - Заголовок/Название панели. В него можно "положить" как обычный текст, так и html (ниже есть пример).

`AccordionContent` - Контент панели. В него можно "положить" как обычный текст, так и html.

<br />

> **\*** У `Accordion` можно указать атрибут `value` со значением id панели, которая будет раскрыта.

> **Примечание!**<br />
> Если у `Accordion` указать атрибут `multiple`, то можно открыть все панели сразу.

> **\*\*** `value` - необходимо указывать, чтобы все панели были закрыты при первой отрисовке.<br />
> Если этот атрибут не указывать, то все панели будут раскрыты.

<br />

## Аккордеон с 1-й панелью

```html dark
<Accordion>
  <AccordionPanel value="1">
    <AccordionHeader>Заголовок панели</AccordionHeader>
    <AccordionContent>
      Контент панели
      <br>
      Lorem ipsum dolor sit amet, consectetur adipisicing elit.
      <br>
      Delectus, saepe.
    </AccordionContent>
  </AccordionPanel>
</Accordion>
```

## Аккордеон с одной выключенной панелью

```html dark
<Accordion>
  <AccordionPanel value="1">
    <AccordionHeader>Заголовок панели #1</AccordionHeader>
    <AccordionContent>
      Контент панели<br>
      Lorem ipsum dolor sit amet, consectetur adipisicing elit. Delectus, saepe.
    </AccordionContent>
  </AccordionPanel>
  <AccordionPanel value="2" disabled>
    <AccordionHeader>Заголовок панели #2</AccordionHeader>
    <AccordionContent>
      Контент панели<br>
      Lorem ipsum dolor sit amet, consectetur adipisicing elit. Delectus, saepe.
    </AccordionContent>
  </AccordionPanel>
</Accordion>
```

> За выключение отвечает атрибут `disabled` у элемента панели `AccordionPanel`.
