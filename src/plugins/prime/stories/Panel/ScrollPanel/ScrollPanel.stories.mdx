## Описание

ScrollPanel - это компонент для создания кастомной области прокрутки с стилизованными скроллбарами. 

### Особенности:

- **Вертикальная прокрутка** - для длинного контента
- **Горизонтальная прокрутка** - для широкого контента
- **Двунаправленная прокрутка** - для контента, который превышает размеры по обеим осям
- **Кастомный дизайн скроллбаров** - стилизованные полосы прокрутки

### Стилизация:

- Размер скроллбара настраивается через токен `{media.size.xs}`
- Скругление углов через `{media.borderRadius.xs}`
- Цвет скроллбара через `{surface.300}` (светлая тема)
- Плавная анимация через `{media.transitionDuration}`

### Использование:

Задайте фиксированные размеры контейнеру через `style` или `class`, и ScrollPanel автоматически добавит прокрутку, если контент превышает эти размеры.
