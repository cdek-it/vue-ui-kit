# TieredMenu

[PrimeVue TieredMenu](https://primevue.org/tieredmenu), [Макет](<https://www.figma.com/design/4TYeki0MDLhfPGJstbIicf/UI-kit-PrimeFace--DS-?node-id=888-12975&m=dev>)

TieredMenu displays submenus in nested overlays.

### Basic

Basic usage with nested menu items without selected state.

```jsx dark
const items = [
  {
    label: 'Home',
    icon: 'ti ti-user',
    items: [
      { label: 'Components' },
      { label: 'Blocks' },
      { label: 'UI Kit' },
      {
        label: 'Templates',
        items: [
          { label: 'Apollo' },
          { label: 'Ultima' }
        ]
      }
    ]
  },
  { label: 'Features' },
  {
    label: 'Projects',
    items: [
      { label: 'Components' },
      { label: 'Blocks' }
    ]
  }
];

<TieredMenu :model="items" />
```

### With Selected Item

Menu with a selected item highlighted using the `p-menuitem-checked` class.

```jsx dark
const items = [
  {
    label: 'Home',
    icon: 'ti ti-user',
    class: 'p-menuitem-checked',
    items: [...]
  }
];

<TieredMenu :model="items" />
```
