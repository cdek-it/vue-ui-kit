### With Selected Item

Меню с выделенным активным пунктом.

<details>
<summary>Посмотреть пример кода выделенного пункта</summary>

Для выделения пункта добавьте `class: 'p-menuitem-checked'` в объект модели:

```js
const items = ref([
 {
  label: 'Home',
  class: 'p-menuitem-checked', // ← активный пункт
 }
]);
```

`p-menuitem-checked` — CSS-класс PrimeVue, который задаёт стиль выбранного состояния. Передаётся через поле `class` в элементе модели меню — PrimeVue автоматически применяет его к `<li>` пункта.

</details>
