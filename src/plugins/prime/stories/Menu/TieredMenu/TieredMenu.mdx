## Варианты использования

### Basic

Базовый вариант меню с вложенными подменю без выделенного пункта.

<Canvas of={TieredMenuStories.Basic} />

### With Selected Item

<details>
<summary>Посмотреть пример кода выделенного пункта</summary>

<div style={{ margin: '-20px 0' }}>

Меню с динамически изменяемым активным пунктом. При клике по пункту меню класс `p-menuitem-checked` переносится на выбранный элемент.

```js
const items = ref([
 {
  label: 'Home',
  class: 'p-menuitem-checked', // ← активный пункт
 }
]);
```

`p-menuitem-checked` — CSS-класс PrimeVue, который задаёт стиль выбранного состояния. Передаётся через поле `class` в элементе модели меню — PrimeVue автоматически применяет его к `<li>` пункта. Переключение реализуется через обработчики `command` в модели.

</div>

</details>
