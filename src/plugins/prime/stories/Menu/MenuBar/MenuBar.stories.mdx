# MenuBar

[PrimeVue Menubar](https://primevue.org/menubar)

Компонент Menubar отображает горизонтальное меню навигации.

## Basic

Базовое горизонтальное меню. По умолчанию все пункты не выбраны. При клике на пункт меню ему присваивается класс `p-menuitem-checked`.

[PrimeVue Basic Example](https://primevue.org/menubar/#basic)

```jsx dark
const items = [
  {
    label: 'File',
    icon: 'ti ti-file',
    items: [
      {
        label: 'New',
        icon: 'ti ti-plus',
      }
    ]
  }
];

<Menubar :model="items" />
```

## Особенности стилизации

### Design Tokens
Основные стили определены в `src/plugins/prime/theme3.0/components/index.ts` с использованием `colorScheme` для разделения цветовых значений.

### Дополнительные CSS
Дополнительные стили в `src/plugins/prime/theme3.0/components/css/menubar.ts`:
- Caption стили
- Active state стили
- Submenu label стили

## Состояния

- **Default** - обычное состояние
- **Hover** - при наведении
- **Selected** - активный элемент, класс `p-menuitem-checked`
- **Disabled** - неактивный элемент
