## Accessability 
По дизайну должна быть возможность управлять чипами с клавиатуры. Tab - может сфокурисроватьна чипе, enter или backspace удалить сфокусированный.
Чтобы эта фича работала, надо руками подкидывать tabindex="0" В компонент.

```html dark
<PrimeChip label="Chip" removable icon="ti ti-info-circle" :pt="{removeIcon: {tabindex: "0"}}" />
```

<Story name="Accessability" args={{label: "Chip", icon: "ti ti-info-circle", removable: true, pt: {removeIcon: {tabindex: "0"}}}}>{Template.bind({})}</Story>

При этом, если вы накидываете disabled, надо руками управлять tabindex. Ниже пример, без обработки дизаблед.

```html dark
<PrimeChip label="Chip" removable icon="ti ti-info-circle" disabled :pt="{removeIcon: {tabindex: "0"}}" />
```

<Story name="AccessabilityDisabledError" args={{label: "Chip", icon: "ti ti-info-circle", removable: true, disabled: true, pt: {removeIcon: {tabindex: "0"}}}}>{Template.bind({})}</Story>

Нужно делать что-то типа


```html dark
<script  setup lang="ts">

const myDisabled = ref(true)
const myTabIndex = computed(() => {
    return myDisabled.value ? "-1" : "0"
})

</script>

<PrimeChip label="Chip" removable icon="ti ti-info-circle" disabled :pt="{removeIcon: {tabindex: myTabIndex}}" />
```
