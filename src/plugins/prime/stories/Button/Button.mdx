# Button

[PrimeVue Button](https://primevue.org/button), [Figma Design](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=160-5223)

## Варианты кнопок

Компонент поддерживает следующие варианты согласно дизайну из Figma:

* **Варианты (severity)**: `primary` (default), `secondary`, `contrast`, `text`, `link`
* **Размеры (size)**: `small`, `base` (default), `large`, `xlarge`
* **Состояния**: `default`, `hover`, `focus`, `active`, `disabled`, `loading`
* **Форма**: обычная или `rounded`
* **С иконкой**: `icon-only`, `prefix`, `postfix`
