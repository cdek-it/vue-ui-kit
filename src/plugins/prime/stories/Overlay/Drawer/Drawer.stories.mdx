# Drawer

### Basic

```html dark
<Button label="Show Drawer" @click="visible = true" />
<Drawer v-model:visible="visible">
  <template #header>
    <h3>Drawer Header</h3>
  </template>
  <p>Drawer content goes here...</p>
</Drawer>
```

### Without Modal

Drawer can be displayed without modal overlay.

```html dark
<Drawer v-model:visible="visible" :modal="false">
  <template #header>
    <h3>Non-Modal Drawer</h3>
  </template>
  <p>This drawer doesn't block interaction with the background.</p>
</Drawer>
```

  name="Without Modal"

### Dismissible

Control whether the drawer can be dismissed by clicking outside or pressing Escape.

```html dark
<Drawer v-model:visible="visible" :dismissible="false">
  <template #header>
    <h3>Non-Dismissible Drawer</h3>
  </template>
  <p>This drawer can only be closed using the close button.</p>
</Drawer>
```
