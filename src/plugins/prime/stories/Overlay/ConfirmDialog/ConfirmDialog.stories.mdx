# ConfirmDialog

[PrimeVue ConfirmDialog](https://primevue.org/confirmdialog)

Компонент ConfirmDialog используется для подтверждения действий пользователя с различными размерами и severity.

## Default

Диалог подтверждения по умолчанию с размером `extra`.

```html dark
<script setup>
import { useConfirm } from 'primevue/useconfirm';

const confirm = useConfirm();

const showConfirm = () => {
  confirm.require({
    message: 'Вы уверены, что хотите продолжить?',
    header: 'Подтверждение',
    icon: 'ti ti-alert-triangle',
    acceptLabel: 'Да',
    rejectLabel: 'Нет'
  });
};
</script>

<template>
  <Button @click="showConfirm">Показать диалог</Button>
  <ConfirmDialog />
</template>
```

## Severity: Success

Диалог подтверждения успешного действия с иконкой в хедере.

```html dark
<script setup>
const showConfirm = () => {
  confirm.require({
    group: 'headless',
    message: 'Операция выполнена успешно!',
    header: 'Успех',
    icon: 'ti ti-circle-check',
    acceptLabel: 'OK',
    rejectLabel: 'Отмена',
    acceptProps: { severity: 'success' },
    rejectProps: { severity: 'secondary', text: true }
  });
};
</script>

<template>
  <ConfirmDialog group="headless" class="p-confirm-dialog-accept">
    <template #container="{ message, acceptCallback, rejectCallback }">
      <div class="p-dialog-header">
        <div class="p-dialog-title">
          <i class="ti ti-circle-check p-icon"></i>
          <span>Успех</span>
        </div>
        <button class="p-dialog-header-close p-button p-button-text p-button-rounded" type="button" @click="rejectCallback">
        </button>
      </div>
      <div class="p-dialog-content">
        <p>{{ message.message }}</p>
      </div>
      <div class="p-dialog-footer">
        <Button label="Отмена" severity="secondary" text @click="rejectCallback" />
        <Button label="OK" severity="success" @click="acceptCallback" />
      </div>
    </template>
  </ConfirmDialog>
</template>
```

<Story name="Severity: Success" args={{ header: 'Успех', message: 'Операция выполнена успешно!', icon: 'ti ti-circle-check', acceptLabel: 'OK', acceptSeverity: 'success', class: 'p-confirm-dialog-accept' }}>{Template.bind({})}</Story>
