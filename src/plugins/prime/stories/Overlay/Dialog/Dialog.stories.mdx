# Dialog

[PrimeVue Dialog](https://primevue.org/dialog), [Figma Design](<https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=520-6620>)

## Описание

Dialog - модальное окно для отображения контента поверх основного интерфейса.

### Типографика

Согласно дизайн-токенам:

**Title:**
- **Font Family**: TT Fellows (heading)
- **Font Size**: 20px (1.25rem)
- **Font Weight**: 600 (demibold)
- **Line Height**: 1.5714285714285714rem

**Content:**
- **Font Family**: PT Sans (base)
- **Font Size**: 16px (1rem)
- **Font Weight**: 400 (regular)
- **Line Height**: 1.5rem

### Размеры

- **Default**: min-width 24rem, max-width 30rem
- **Small**: min-width 20rem, max-width 24rem
- **Large**: min-width 30rem, max-width 58rem
- **Extra Large**: min-width 58rem, max-width 90vw
