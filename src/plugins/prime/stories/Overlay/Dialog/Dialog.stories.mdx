# Dialog

[PrimeVue Dialog](https://primevue.org/dialog)

Компонент Dialog отображает модальное окно с различными размерами.

## Default (Extra)

Размер по умолчанию с `min-width` и `max-width` из `sizingDialog.extra`.

```html dark
<Dialog 
  header="Заголовок диалога" 
  modal 
  v-model:visible="visible"
  class="p-dialog-extra"
>
  <div>Содержимое диалога</div>
  <template #footer>
    <Button label="Отмена" text" />
    <Button label="Сохранить" />
  </template>
</Dialog>
```
