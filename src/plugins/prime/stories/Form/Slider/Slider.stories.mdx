# Slider

[PrimeVue Slider](https://primevue.org/slider), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=484-6137)

Slider - это компонент для выбора числового значения или диапазона значений путем перемещения ползунка.

## Базовый слайдер

Простой слайдер для выбора одного значения от 0 до 100.

```html dark
<script setup>
import { ref } from 'vue';
const value = ref(50);
</script>

<template>
  <PrimeSlider v-model="value" />
</template>
```

<Story name="Basic">{BasicTemplate.bind({})}</Story>

## Range слайдер

Выбор диапазона значений с двумя ползунками.

```html dark
<script setup>
import { ref } from 'vue';
const rangeValue = ref([20, 80]);
</script>

<template>
  <PrimeSlider v-model="rangeValue" range />
</template>
```

<Story name="Range">{RangeTemplate.bind({})}</Story>

## Слайдер с шагом

Слайдер с заданным шагом изменения значения (например, шаг 10: 0, 10, 20, 30...).

```html dark
<script setup>
import { ref } from 'vue';
const value = ref(50);
</script>

<template>
  <PrimeSlider v-model="value" :step="10" />
</template>
```

<Story name="Step">{StepTemplate.bind({})}</Story>

## Слайдер с min/max

Ограничение диапазона значений (от 20 до 80).

</script>

<template>

## Токены стилизации

### Track (дорожка)
- `slider.track.background` → `{content.borderColor}` - цвет неактивной части
- `slider.track.borderRadius` → `{content.borderRadius}` → `{borderRadius.md}` = `0.75rem`
- `slider.track.size` → `{form.size.xs}` → `{sizing.1}` = `0.25rem`

### Range (активная часть)
- `slider.range.background` → `{surface.900}` - цвет активной части

### Handle (ползунок)
- `slider.handle.width` → `{form.size.xlg}` → `{sizing.6}` = `1.5rem`
- `slider.handle.height` → `{form.size.xlg}` → `{sizing.6}` = `1.5rem`
- `slider.handle.borderRadius` → `{form.borderRadius.max}` → `{borderRadius.max}` = `999rem`
- `slider.handle.background` → `{surface.900}` - цвет ползунка
- `slider.handle.content.width` → `{form.size.md}` → `{sizing.3}` = `0.75rem` - внутренний круг
- `slider.handle.content.height` → `{form.size.md}` → `{sizing.3}` = `0.75rem`
- `slider.handle.content.background` → `{surface.0}` - цвет внутреннего круга

### Анимация
- `slider.root.transitionDuration` → `{form.transitionDuration}` → `{transition.duration.200}` = `200ms`
