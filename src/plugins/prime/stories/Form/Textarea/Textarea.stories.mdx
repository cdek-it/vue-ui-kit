# Textarea

[PrimeVue Textarea](https://primevue.org/textarea), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=309-1582&t=j8q05RFyXpKxntpS-11)

## Basic States

```html dark
<Textarea v-model="value" placeholder="Placeholder" rows="6" />
```

## Sizes

Textarea поддерживает три размера: `small`, default (medium), и `large`.

```html dark
<Textarea v-model="value" placeholder="Placeholder" rows="6" size="small" />
<Textarea v-model="value" placeholder="Placeholder" rows="6" />
<Textarea v-model="value" placeholder="Placeholder" rows="6" size="large" />
```

<Story name="Sizes">{TemplateSizes.bind({})}</Story>

## Readonly

Состояние `readonly` позволяет отображать текст без возможности редактирования.

```html dark
<Textarea v-model="value" placeholder="Placeholder" rows="6" readonly />
```

## With Clear Icon

Textarea с кнопкой удаления текста. Иконка автоматически скрывается, когда поле пустое.

```html dark
<IconField>
  <Textarea v-model="value" placeholder="Placeholder" rows="6" />
  <InputIcon @click="clearValue">
    <i class="ti ti-x" />
  </InputIcon>
</IconField>
```
