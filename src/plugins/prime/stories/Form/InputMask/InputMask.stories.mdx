# InputMask

[PrimeVue InputMask](https://primevue.org/inputmask), [Макет](<https://www.figma.com/design/4TYeki0MDLhfPGJstbIicf/UI-kit-PrimeFace-(DS)?node-id=484-5521&node-type=section&t=2PSg63p2UfJ4cUzB-0>)

InputMask компонент для ввода данных с маской форматирования.

## Базовые состояния

```html dark
<PrimeInputMask mask="99-999999" placeholder="99-999999" />
<PrimeInputMask mask="99-999999" placeholder="99-999999" invalid />
<PrimeInputMask mask="99-999999" placeholder="99-999999" disabled />
<PrimeInputMask mask="99-999999" placeholder="99-999999" readonly />
```

## С иконкой

InputMask с иконкой закрытия. Иконка появляется только когда введен хотя бы 1 символ.

```html dark
<PrimeIconField>
  <PrimeInputMask v-model="inputValue" mask="99-999999" placeholder="99-999999" />
  <PrimeInputIcon @click="onClickDelete">
    <i class="ti ti-x" />
  </PrimeInputIcon>
</PrimeIconField>
```

## Примеры масок

Различные форматы масок для разных типов данных.

```html dark
<PrimeInputMask mask="99-999999" placeholder="Phone: 99-999999" />
<PrimeInputMask mask="99.99.9999" placeholder="Date: dd.mm.yyyy" />
<PrimeInputMask mask="(999) 999-9999" placeholder="Phone US: (999) 999-9999" />
<PrimeInputMask mask="99:99" placeholder="Time: 99:99" />
<PrimeInputMask mask="a*-999-a999" placeholder="Serial: a*-999-a999" />
```
