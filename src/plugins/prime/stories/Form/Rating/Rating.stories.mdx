# Rating

[PrimeVue Rating](https://primevue.org/rating)

Компонент Rating используется для оценки с помощью звезд.

## Default

Базовый компонент рейтинга.

```html dark
<script setup>
import { ref } from 'vue';

const value = ref(null);
</script>

<template>
  <Rating v-model="value" />
</template>
```

<Story name="Default">{Template.bind({})}</Story>

## With Value

Рейтинг с предустановленным значением.

```html dark
<Rating v-model="value" />
```

<Story name="With Value" args={{ modelValue: 3 }}>{Template.bind({})}</Story>

## Read Only

Рейтинг только для чтения.

```html dark
<Rating v-model="value" :readonly="true" />
```

<Story name="Read Only" args={{ modelValue: 4, readonly: true }}>{Template.bind({})}</Story>

## Disabled

Отключенный рейтинг.

```html dark
<Rating v-model="value" :disabled="true" />
```

<Story name="Disabled" args={{ modelValue: 2, disabled: true }}>{Template.bind({})}</Story>

## Without Cancel

Рейтинг без возможности отмены выбора.

```html dark
<Rating v-model="value" :cancel="false" />
```

<Story name="Without Cancel" args={{ modelValue: 3, cancel: false }}>{Template.bind({})}</Story>

## Custom Stars

Рейтинг с пользовательским количеством звезд.

```html dark
<Rating v-model="value" :stars="10" />
```

<Story name="Custom Stars" args={{ modelValue: 7, stars: 10 }}>{Template.bind({})}</Story>
