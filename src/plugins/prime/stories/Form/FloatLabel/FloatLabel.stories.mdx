# FloatLabel

### Input

Какие пропсы передавать в инпут см. [InputText](/docs/prime-form-inputtext--primary)

```html dark
<PrimeFloatLabel variant="in">
  <PrimeInputText id="input" />
  <label for="input">FloatLabel</label>
</PrimeFloatLabel>
```

### With Icon

FloatLabel с иконкой закрытия. Иконка появляется только когда введен хотя бы 1 символ.

```html dark
<PrimeFloatLabel variant="in">
  <PrimeIconField>
    <PrimeInputText id="username" v-model="inputValue" />
    <PrimeInputIcon @click="onClickDelete">
      <i class="ti ti-x" />
    </PrimeInputIcon>
  </PrimeIconField>
  <label for="username">Username</label>
</PrimeFloatLabel>
```
