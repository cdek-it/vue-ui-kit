## С иконками

Логика на работу крестика не прописана в ui-ките, на данный момент должна быть прописана на стороне приложения.
Иконка появляется только когда в поле введен хотя бы один символ.

```html dark
<script setup>
  const inputValue = ref('');

  const onClickDelete = () => {
    inputValue.value = '';
  };
</script>

<template>
  <PrimeIconField>
    <PrimeInputText v-model="inputValue" placeholder="Normal" />
    <PrimeInputIcon v-if="inputValue.length > 0" @click="onClickDelete">
      <i class="ti ti-x" />
    </PrimeInputIcon>
  </PrimeIconField>
</template>
```

## FloatLabel

Плавающая метка для InputText. Поддерживает три варианта позиционирования.

### Variant: Inside (IFTA)

Метка находится внутри поля и поднимается вверх при фокусе или наличии значения.

```html dark
<PrimeFloatLabel variant="inside">
  <PrimeInputText id="username" v-model="value" />
  <label for="username">FloatLabel</label>
</PrimeFloatLabel>
```

<Story name="FloatLabel">{TemplateFloatLabel.bind({})}</Story>

### Variant: On

Метка появляется на границе поля при фокусе или наличии значения.

```html dark
<PrimeFloatLabel variant="on">
  <PrimeInputText id="username" v-model="value" />
  <label for="username">FloatLabel</label>
</PrimeFloatLabel>
```

<Story name="FloatLabelOn">{TemplateFloatLabelOn.bind({})}</Story>

### Variant: Over

Метка находится над полем и поднимается выше при фокусе или наличии значения.

```html dark
<PrimeFloatLabel variant="over">
  <PrimeInputText id="username" v-model="value" />
  <label for="username">FloatLabel</label>
</PrimeFloatLabel>
```

<Story name="FloatLabelOver">{TemplateFloatLabelOver.bind({})}</Story>

## FloatLabel с иконками

Комбинация FloatLabel и IconField.
Иконка очистки появляется только когда есть значение в поле.

```html dark
<script setup>
  const inputValue = ref('InputText');

  const onClickDelete = () => {
    inputValue.value = '';
  };
</script>

<template>
  <PrimeFloatLabel>
    <PrimeIconField>
      <PrimeInputText id="float-icon" v-model="inputValue" />
      <PrimeInputIcon v-if="inputValue.length > 0" @click="onClickDelete">
        <i class="ti ti-x" />
      </PrimeInputIcon>
    </PrimeIconField>
    <label for="float-icon">FloatLabel</label>
  </PrimeFloatLabel>
</template>
```
