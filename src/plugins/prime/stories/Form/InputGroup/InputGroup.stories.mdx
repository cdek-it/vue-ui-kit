# InputGroup

[PrimeVue InputGroup](https://primevue.org/inputgroup), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=437-3505)

InputGroup позволяет добавлять иконки или текст до и после поля ввода.

## Базовые примеры

```html dark
<!-- Левый addon -->
<PrimeInputGroup>
  <PrimeInputGroupAddon>
    <i class="ti ti-user"></i>
  </PrimeInputGroupAddon>
  <PrimeInputText placeholder="Placeholder" />
</PrimeInputGroup>

<!-- Правый addon -->
<PrimeInputGroup>
  <PrimeInputText placeholder="Placeholder" />
  <PrimeInputGroupAddon>
    <i class="ti ti-user"></i>
  </PrimeInputGroupAddon>
</PrimeInputGroup>

<!-- Оба addon -->
<PrimeInputGroup>
  <PrimeInputGroupAddon>
    <i class="ti ti-user"></i>
  </PrimeInputGroupAddon>
  <PrimeInputText placeholder="Placeholder" />
  <PrimeInputGroupAddon>
    <i class="ti ti-user"></i>
  </PrimeInputGroupAddon>
</PrimeInputGroup>
```

## Состояния

Различные состояния InputGroup с разными позициями addon.

```html dark
<PrimeInputGroup>
  <PrimeInputGroupAddon>
    <i class="ti ti-user"></i>
  </PrimeInputGroupAddon>
  <PrimeInputText placeholder="Placeholder" invalid />
</PrimeInputGroup>
```

## С текстом

InputGroup может содержать текстовые addon вместо иконок.

```html dark
<PrimeInputGroup>
  <PrimeInputGroupAddon>@</PrimeInputGroupAddon>
  <PrimeInputText placeholder="Username" />
</PrimeInputGroup>

<PrimeInputGroup>
  <PrimeInputText placeholder="Price" />
  <PrimeInputGroupAddon>$</PrimeInputGroupAddon>
</PrimeInputGroup>

<PrimeInputGroup>
  <PrimeInputGroupAddon>https://</PrimeInputGroupAddon>
  <PrimeInputText placeholder="example.com" />
  <PrimeInputGroupAddon>.com</PrimeInputGroupAddon>
</PrimeInputGroup>
```

## Множественные addon

Можно добавить несколько addon с одной стороны.

```html dark
<PrimeInputGroup>
  <PrimeInputGroupAddon>
    <i class="ti ti-user"></i>
  </PrimeInputGroupAddon>
  <PrimeInputGroupAddon>
    <i class="ti ti-mail"></i>
  </PrimeInputGroupAddon>
  <PrimeInputText placeholder="Placeholder" />
</PrimeInputGroup>
```
