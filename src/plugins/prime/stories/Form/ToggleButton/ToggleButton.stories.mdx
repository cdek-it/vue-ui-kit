# ToggleButton

[PrimeVue ToggleButton](https://primevue.org/togglebutton/), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=174-1363&t=j8q05RFyXpKxntpS-11)

### Basic

```html dark
<PrimeToggleButton
    v-model="value"
    off-icon="ti ti-arrow-down-right"
    on-icon="ti ti-arrow-down-right"
    off-label="ToggleButton"
    on-label="ToggleButton"
/>
```

### Слот

В макетах скорее всего нет изменения текста, поэтому предлагаю передавать текст и икону через дефолтный слот. Обрати внимание, что при передаче дефолтного слота, иконочный слот работать не будет

```html dark
<ToggleButton>
    <div>
        <i class="ti ti-arrow-down-right"/> Дефолтный слот
    </div>
</ToggleButton>
```

### Слот иконки

```html dark
<ToggleButton off-label="ToggleButton" on-label="ToggleButton">
     <template #icon>
         <i class="ti ti-ban"/>
     </template>
</ToggleButton>
```
