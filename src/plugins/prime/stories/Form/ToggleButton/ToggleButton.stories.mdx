# ToggleButton

[PrimeVue ToggleButton](https://primevue.org/togglebutton/), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/%F0%9F%A7%A9-UI-kit-PrimeFace--1.0.0-beta-?node-id=174-1363)

### Основные варианты

Компонент поддерживает различные размеры (small, base, large, xlarge), состояния (checked/unchecked, disabled), а также позиции иконок (left, right, icon-only).

```html dark
<PrimeToggleButton
    v-model="value"
    off-icon="ti ti-arrow-down-right"
    on-icon="ti ti-arrow-down-right"
    off-label="ToggleButton"
    on-label="ToggleButton"
/>
```

### Размеры

Доступны 4 размера:
- `size="small"` - маленький размер
- По умолчанию - базовый размер
- `size="large"` - большой размер
- `:pt="{ root: { class: 'p-togglebutton-xlarge' } }"` - очень большой размер

### Иконки

Иконки можно размещать слева (по умолчанию), справа (`icon-pos="right"`), или использовать только иконку без текста (`:pt="{ root: { class: 'p-togglebutton-icon-only' } }"`).

```html dark
<!-- Иконка слева -->
<ToggleButton 
    v-model="checked" 
    off-icon="ti ti-arrow-down-right" 
    on-icon="ti ti-arrow-down-right"
    off-label="ToggleButton" 
    on-label="ToggleButton" 
/>

<!-- Иконка справа -->
<ToggleButton 
    v-model="checked" 
    off-icon="ti ti-arrow-down-right" 
    on-icon="ti ti-arrow-down-right"
    icon-pos="right"
    off-label="ToggleButton" 
    on-label="ToggleButton" 
/>

<!-- Только иконка -->
<ToggleButton 
    :pt="{ root: { class: 'p-togglebutton-icon-only' } }"
    v-model="checked" 
    off-icon="ti ti-arrow-down-right" 
    on-icon="ti ti-arrow-down-right"
/>
```

### Слот

В макетах скорее всего нет изменения текста, поэтому предлагаю передавать текст и икону через дефолтный слот. Обрати внимание, что при передаче дефолтного слота, иконочный слот работать не будет

```html dark
<ToggleButton>
    <div>
        <i class="ti ti-arrow-down-right"/> Дефолтный слот
    </div>
</ToggleButton>
```
