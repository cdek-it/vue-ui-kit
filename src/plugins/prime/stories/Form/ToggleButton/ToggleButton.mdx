# Rating

[PrimeVue ToggleButton](https://primevue.org/togglebutton/), [Макет](https://www.figma.com/design/4TYeki0MDLhfPGJstbIicf/UI-kit-PrimeFace-\(DS\)?node-id=484-4821\&t=3YXMhb09kaDg7lEL-4)

### Basic

```html dark
<PrimeToggleButton
    v-model="value"
    off-icon="ti ti-arrow-down-right"
    on-icon="ti ti-arrow-down-right"
    off-label="ToggleButton"
    on-label="ToggleButton"
/>
```

### Слот

В макетах скорее всего нет изменения текста, поэтому предлагаю передавать текст и икону через дефолтный слот. Обрати внимание, что при передаче дефолтного слота, иконочный слот работать не будет

```html dark
<ToggleButton>
    <div>
        <i class="ti ti-arrow-down-right"/> Дефолтный слот
    </div>
</ToggleButton>
```
