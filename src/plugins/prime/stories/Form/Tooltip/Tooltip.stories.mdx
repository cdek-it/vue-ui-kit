# Tooltip

[PrimeVue Tooltip](https://primevue.org/tooltip)

Компонент Tooltip отображает всплывающую подсказку при наведении на элемент.

## Default

Базовая подсказка.

```html dark
<Button label="Наведите на меня" v-tooltip="'Это подсказка'" />
```

<Story name="Default">{Template.bind({})}</Story>

## Top Position

Подсказка сверху.

```html dark

```
