# Badge

[PrimeVue Badge](https://primevue.org/badge), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=21-1193)

Badge компонент используется для отображения статусов, уведомлений и меток.

## Severity варианты

Доступные варианты: `primary`, `secondary`, `success`, `info`, `warn`, `danger`

```html dark
<PrimeBadge value="Badge" severity="primary" />
<PrimeBadge value="Badge" severity="info" />
<PrimeBadge value="Badge" severity="success" />
<PrimeBadge value="Badge" severity="warn" />
<PrimeBadge value="Badge" severity="danger" />
```

<Story of={BadgeStories.Default} />

## Размеры

Доступные размеры: `base` (по умолчанию), `large`, `xlarge`

```html dark
<PrimeBadge value="Badge" />
<PrimeBadge value="Badge" size="large" />
<PrimeBadge value="Badge" size="xlarge" />
```

<Story of={BadgeStories.Sizes} />

## Dot вариант

Badge может отображаться как точка без текста. Используется для индикаторов статуса.

```html dark
<PrimeBadge severity="primary" />
<PrimeBadge severity="info" />
<PrimeBadge severity="success" />
<PrimeBadge severity="warn" />
<PrimeBadge severity="danger" />
```
