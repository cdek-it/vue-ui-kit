# Galleria
[PrimeVue Galleria](https://primevue.org/galleria/)

Galleria - это расширенный компонент для отображения изображений в виде галереи с поддержкой миниатюр, индикаторов, навигации и полноэкранного режима.

## Galleria с индикаторами

Базовый пример галереи с индикаторами и навигационными кнопками.

```html dark
<Galleria 
  :value="images" 
  :numVisible="5"
  containerStyle="max-width: 640px"
  :showItemNavigators="true"
  :showIndicators="true"
>
  <template #item="slotProps">
    <img :src="slotProps.item.itemImageSrc" :alt="slotProps.item.alt" style="width: 100%; display: block;" />
  </template>
  <template #thumbnail="slotProps">
    <img :src="slotProps.item.thumbnailImageSrc" :alt="slotProps.item.alt" style="display: block;" />
  </template>
</Galleria>
```

## Структура данных

```ts dark
const images = [
  {
    itemImageSrc: 'path/to/image1.jpg',
    thumbnailImageSrc: 'path/to/thumbnail1.jpg',
    alt: 'Description for Image 1',
    title: 'Title 1'
  },
  // ...
];
```

## Основные свойства

- **value** - массив объектов с данными изображений
- **numVisible** - количество видимых миниатюр
- **showItemNavigators** - показывать кнопки навигации
- **showIndicators** - показывать индикаторы
- **showThumbnails** - показывать миниатюры
- **fullScreen** - режим полного экрана
- **autoPlay** - автоматическое воспроизведение
- **transitionInterval** - интервал смены изображений (мс)
- **circular** - циклическая прокрутка
