# Skeleton

[PrimeVue Skeleton](https://primevue.org/skeleton), [Макет](<https://www.figma.com/design/4TYeki0MDLhfPGJstbIicf/UI-kit-PrimeFace--DS-?node-id=5241-3731&m=dev>)

Компонент Skeleton используется как placeholder для загружаемого контента.

## Default Rectangle

Базовый прямоугольный скелетон.

```html dark
<Skeleton />
```

<Story name="Default">{Template.bind({})}</Story>

## Small Rectangle

Маленький прямоугольный скелетон.

```html dark

```

## Card Layout

Пример скелетона для карточки с аватаром и текстом.

```html dark
<div style="display: flex; gap: 1rem; align-items: center">
  <Skeleton shape="circle" size="4rem" />
  <div style="flex: 1">
    <Skeleton width="100%" height="1rem" style="margin-bottom: 0.5rem" />
    <Skeleton width="75%" height="1rem" />
  </div>
</div>
```
