## Описание

Skeleton используется для отображения заглушки контента во время загрузки данных. Компонент помогает улучшить восприятие производительности приложения.

### Формы:

- **Rectangle** (по умолчанию) - прямоугольная форма
- **Circle** - круглая форма (используйте `shape="circle"`)
- **Custom** - кастомная форма через `borderRadius`

### Анимация:

- **Wave** (по умолчанию) - волновая анимация
- **None** - без анимации (используйте `animation="none"`)

### Токены стилизации:

- `skeleton.root.borderRadius` → `{content.borderRadius}` → `{borderRadius.md}` = `0.75rem`
- `skeleton.colorScheme.light.root.background` → `{surface.200}` - основной цвет фона
- `skeleton.colorScheme.light.root.animationBackground` → `{surface.100}` - цвет анимации

### Примеры использования:

#### Базовый скелетон
```html
<PrimeSkeleton width="10rem" height="4rem" />
```

#### Круглый скелетон (аватар)
```html
<PrimeSkeleton shape="circle" size="4rem" />
```

#### Без анимации
```html
<PrimeSkeleton width="10rem" height="4rem" animation="none" />
```

#### Карточка профиля
```html
<div class="card">
  <div class="flex gap-3 mb-4">
    <PrimeSkeleton shape="circle" size="4rem" />
    <div class="flex-1">
      <PrimeSkeleton width="70%" height="1.5rem" class="mb-2" />
      <PrimeSkeleton width="50%" height="1rem" />
    </div>
  </div>
  <PrimeSkeleton width="100%" height="1rem" class="mb-2" />
  <PrimeSkeleton width="100%" height="1rem" class="mb-2" />
  <PrimeSkeleton width="80%" height="1rem" />
</div>
```
