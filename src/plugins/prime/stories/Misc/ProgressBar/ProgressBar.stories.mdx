# ProgressBar

[PrimeVue ProgressBar](https://primevue.org/progressbar), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=113-1004)

### Base

Базовый прогресс-бар с процентами

```html dark
<PrimeProgressBar :value="50" />
```

<Story name="Primary" args={{ value: 50 }}>
  {Template.bind({})}
</Story>

### Без значения (Indeterminate)

Прогресс-бар без конкретного значения для неопределенных процессов

```html dark
<PrimeProgressBar mode="indeterminate" />
```

<Story name="Indeterminate" args={{ mode: 'indeterminate' }}>
  {Template.bind({})}
</Story>
