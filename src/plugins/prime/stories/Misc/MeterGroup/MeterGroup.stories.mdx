# MeterGroup

[PrimeVue MeterGroup](https://primevue.org/metergroup)

Компонент MeterGroup отображает несколько значений в виде сегментированной полосы прогресса.

## Default

Базовый MeterGroup с несколькими значениями и цветами из палитры severity.

```html dark
<script setup>
const value = [
  { label: 'Success', color: '#44e858', value: 25 },  // success.500
  { label: 'Info', color: '#4496e8', value: 15 },     // info.500
  { label: 'Warn', color: '#f5b83d', value: 20 },     // warn.500
  { label: 'Error', color: '#e85244', value: 10 }     // error.500
];
</script>

<template>
  <MeterGroup :value="value" />
</template>
```
