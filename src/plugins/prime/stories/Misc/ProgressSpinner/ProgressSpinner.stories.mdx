# ProgressSpinner

[PrimeVue ProgressSpinner](https://primevue.org/progressspinner), [Макет](<https://www.figma.com/design/4TYeki0MDLhfPGJstbIicf/UI-kit-PrimeFace--DS-?node-id=1219-3086&m=dev>)

Компонент ProgressSpinner отображает анимированный индикатор загрузки с четырьмя цветами.

## Default

Базовый спиннер с цветами из дизайн-токенов: success (зеленый), info (синий), error (красный), warn (желтый).

```html dark
<ProgressSpinner />
```

<Story name="Default">{Template.bind({})}</Story>
