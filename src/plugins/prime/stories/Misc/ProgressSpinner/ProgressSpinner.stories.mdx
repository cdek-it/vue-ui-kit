# ProgressSpinner

[PrimeVue ProgressSpinner](https://primevue.org/progressspinner), [Макет](https://www.figma.com/design/HOLKdvQJ8jCLeX17s9d0Yf/UI-Kit--DS--v2.0?node-id=1219-3101)

### Base

Базовый спиннер (48px)

```html dark
<PrimeProgressSpinner />
```
### Размеры

Различные размеры спиннера согласно дизайн-системе

```html dark
<!-- XLarge: 64px -->
<PrimeProgressSpinner class="p-progressspinner-xlarge" />

<!-- Large: 48px -->
<PrimeProgressSpinner class="p-progressspinner-large" />

<!-- Medium: 32px -->
<PrimeProgressSpinner class="p-progressspinner-medium" />

<!-- Small: 24px -->
<PrimeProgressSpinner class="p-progressspinner-small" />
```
