# Быстрый старт

Пошаговый чеклист для подключения дизайн-системы к Vue-проекту.

## 1. Установка зависимостей

```bash dark
npm install primevue @cdek-ui-kit/vue
```

## 2. Подключение PrimeVue

```js dark
import PrimeVue from 'primevue/config';
import { getPrimeVueConfig } from '@cdek-ui-kit/vue';

app.use(PrimeVue, getPrimeVueConfig());
```

Подробнее: [Установка](?path=/docs/prime-установка--docs)

## 3. Подключение типографики и иконок

Добавьте в `index.html`:

```html dark
<link rel="preconnect" href="https://public-static.cdek.ru" />
<link
  rel="stylesheet"
  href="https://public-static.cdek.ru/common/icons/v3.30.0/tabler-icons.min.css"
/>
<link
  rel="stylesheet"
  href="https://public-static.cdek.ru/common/typography/v1.0.0/typography.min.css"
/>
```

Подробнее: [Типографика](?path=/docs/prime-типографика--docs) | [Иконки](?path=/docs/prime-иконки--docs)

## 4. Tailwind CSS (опционально)

```bash dark
npm install tailwindcss tailwindcss-primeui
```

В `tailwind.config.js`:

```js dark
const { tailwindConfigTheme } = require('@cdek-ui-kit/vue');

export default {
  content: ['./src/**/*.{vue,js,ts}'],
  plugins: [require('tailwindcss-primeui')],
  theme: {
    extend: {
      ...tailwindConfigTheme,
    },
  },
};
```

Подробнее: [Tailwind](?path=/docs/prime-tailwind--docs)

## 5. Первый компонент

```html dark
<template>
  <PrimeButton label="Готово" />
</template>
```

Если используете авто-импорт, компонент доступен сразу. Иначе:

```js dark
import Button from 'primevue/button';
```

## Что входит в DS

- **Тема** — дизайн-токены для всех компонентов PrimeVue (цвета, отступы, скругления, тени)
- **Типографика** — CSS-классы для текста по дизайн-системе
- **Иконки** — иконочный шрифт Tabler Icons
- **Компоненты** — стилизованные компоненты PrimeVue + PBlock-обёртки с бизнес-логикой
- **Tailwind-интеграция** — токены DS доступны как Tailwind-утилиты
