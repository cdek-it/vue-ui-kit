# Типографика

## Подключение

Добавьте в `index.html`:

```html dark
<link rel="preconnect" href="https://public-static.cdek.ru" />
<link
  rel="stylesheet"
  href="https://public-static.cdek.ru/common/typography/v1.0.0/typography.min.css"
/>
```

`typography.min.css` заводит CSS-переменные на основе токенов DS и CSS-классы для каждого вида текста.

## Как найти нужный класс в Figma

Нажмите на компонент `<Typography.*>` (не на сам текст), и вы увидите название класса:

<img src="/assets/typography-font-class.png" />

## SCSS-миксины

Если вам нужно задать типографику в SCSS, используйте миксины из `@/assets/style/vars/fonts.scss`:

```scss dark
@import '@/assets/style/vars/fonts';

.my-title {
  @include headline-4;
}

.my-text {
  @include body-2;
}
```
