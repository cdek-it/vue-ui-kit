# SSR

Тосты должны вызываться только на клиентской части конечного приложения, так как они маунтятся как второе приложение возле исходного.

Хук `useToast` можно использовать на серверной части, таким образом будет происходить инициализация тостов.

Вызов тостов описан в историях.

Также доступны методы для обновления - `update( id: any, settings: ToasterProps)`, выключения конкретного тоста - `dismiss (id: ToastID)`
